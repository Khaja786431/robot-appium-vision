Installation
============

Recommended installation using virtual environment.

Windows
-------

.. code-block:: bash

   python -m venv venv
   .\venv\Scripts\Activate.ps1
   pip install robot-appium-vision

Linux / macOS
-------------

.. code-block:: bash

   python3 -m venv venv
   source venv/bin/activate
   pip install robot-appium-vision
